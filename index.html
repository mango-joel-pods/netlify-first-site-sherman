    Here is the cheat sheet

    1. Install github
    2. open terminal and use the "cd" command to change directories into a repo folder
    3. Initialize the folder into a git repo using "git init"
    4. Tell git about the files using "git add ."
    5. Then, run git status command to see the files that have changed
    6. use "git commit -am 'first commit'"
    7. change to the main branch using "git branch -M main"
    8. add the remote repo as the origin of this branch copy and pasting the github link
    9. push the commit using code "git push origin main"
